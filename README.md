
# Student Course Enrollment System

**Author:** Yashwanth Kumar M V

## Description
This project is a SQL-based mini project to manage students, courses, enrollments, exams, and results.

## Features
- Student & Course Management
- Enrollment Tracking
- Exam & Results Management
- SQL Queries for Analysis

## How to Run
1. Open MySQL Workbench
2. Run `main.sql`
3. Execute queries

## Sample Outputs
- List of students
- Courses
- Top scoring student
- Average marks

## Conclusion
This project demonstrates SQL database design and querying.
